## About

**Snakr** é uma plataforma de **centralização de bibliotecas de jogos**, criada para unificar, organizar e exibir todos os jogos de um usuário em um único lugar, independentemente da plataforma de origem.

A aplicação permite conectar contas de serviços como **Steam, Epic Games e outras**, reunindo jogos, conquistas, status e atividades em uma interface moderna, rápida e focada em experiência do usuário.  
Além disso, o Snakr centraliza **notícias do mundo dos games em tempo real** e oferece um espaço dedicado para **resgate e acompanhamento de jogos, DLCs e conteúdos gratuitos** disponíveis nas principais plataformas.


## Purpose & Vision

- **Propósito:**  
  Eliminar a fragmentação entre plataformas de jogos, oferecendo um **hub único** onde o jogador possa visualizar sua biblioteca completa, acompanhar progresso, consumir notícias relevantes e aproveitar oportunidades de conteúdo gratuito.

- **Visão de Longo Prazo:**  
  - Tornar-se o principal **game hub pessoal** do usuário.  
  - Centralizar bibliotecas, conquistas, tempo de jogo e atividades.  
  - Agregar **notícias atualizadas do ecossistema gamer** em tempo real.  
  - Facilitar o **resgate e o acompanhamento de jogos, DLCs e benefícios gratuitos**.  
  - Oferecer recursos sociais semelhantes a plataformas tradicionais, porém de forma unificada.  
  - Evoluir para um ecossistema extensível com integrações contínuas.


## Technical Architecture

| Camada | Tecnologia / Ferramenta |
|---|---|
| Frontend | React, React Hooks, React Router, CSS customizado |
| Backend | Node.js, API REST |
| Integrações | APIs externas (Steam, NewsApi, etc.) |
| Hospedagem | Vercel |
| Autenticação | OAuth / Tokens de acesso |
| Gerenciamento de estado | Hooks |
| Deploy | CI/CD automatizado |

A arquitetura foi pensada para permitir **novas integrações de plataformas** sem impacto direto na base do sistema.
